# Política de Segurança

A segurança dos dados é tratada com prioridade neste projeto.

---

## Boas Práticas

- Nunca commite segredos (tokens, chaves, arquivos `.env`)
- Variáveis sensíveis devem ser configuradas apenas em:
  - GitHub Secrets
  - Environment Variables da Vercel

---

## Reporte de Vulnerabilidades

Se você identificar uma vulnerabilidade de segurança, **não abra uma issue pública**.

Entre em contato de forma responsável:
- Email: security@fiscalbox.com.br (placeholder)

---

## Escopo

Este projeto:
- Processa dados financeiros e fiscais enviados pelo usuário
- Atua como **ferramenta auxiliar**
- Não executa operações fiscais oficiais nem transmissões governamentais
